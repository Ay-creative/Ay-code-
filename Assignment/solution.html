Web development

HTML

HTML (Hyper text markup language): Html generally considered the easiest of the three. HTML defines the structures and the content of the web pages using tags. Learning html is very straigth forward since it involve understanding how to organize content using element.

CSS

CSS(Cascading style sheets): CSS can be moderate to challenging depending on complexity. CSS is used for styling HTML elements, controlling layout, and presentation on web pages. It involve learning selectors, properties, value, and how the interact to style elements.

JAVA SCRIPT

Java script: Java script often considered the most challenging of the three. Java script is a programming language used create dynamic behaviour on a web pages, such as interactivity, animation, and complex logic. Understanding java script require learning concept like variables, data types, functions, control flow, and object-oriented programming.
